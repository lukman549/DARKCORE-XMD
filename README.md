## üìå Overview
**DARKCORE-XMD** is a fast, session-based **WhatsApp Multi-Device bot** built with **Baileys MD** and **Node.js**.  
It uses **pair-code authentication**, stores sessions locally, and requires **no WhatsApp API tokens**.

---

## ‚ú® Features
- Baileys Multi-Device (MD)
- Pair Code & Session Login
- Group & Owner Commands
- Media / YouTube Downloaders
- Stickers & Anti-Delete
- Auto Typing & Recording
- Plugin-based commands
